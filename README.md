## 不只是"放大"——AI 帧插值也有

很多视频增强工具只做分辨率提升，Nero 还支持 **AI 帧插值（Frame Interpolation）**。

意思是：把原本 24fps 的视频，通过 AI 推算出中间帧，平滑提升到 60fps 甚至 120fps。看起来更流畅，慢动作效果也可以用这个来做。对做短视频、后期剪辑的人来说很实用。
